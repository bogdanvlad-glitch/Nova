# NovaBuild

Site statique multi-pages en HTML, CSS et JavaScript.

## Pages

- `index.html`
- `solutions.html`
- `services.html`
- `contact.html`

## Ouvrir le site

- Soit ouvrir `index.html` dans un navigateur
- Soit lancer un serveur local depuis le dossier:

```bash
python3 -m http.server 8123
```

Puis visiter `http://127.0.0.1:8123`
